## 📌 Project Overview

This project presents a professional HR Analytics Dashboard built using Power BI, designed to help HR managers gain deep insights into workforce data.

The dashboard focuses on understanding employee demographics, performance, recruitment efficiency, and workforce distribution, enabling better HR decision-making.

It demonstrates a complete end-to-end data analytics workflow, including:

Data Cleaning & Transformation
Data Modeling (Star Schema)
KPI Development using DAX
Interactive Dashboard Design

🎯 Goal: Help HR teams monitor workforce trends, improve hiring strategies, and ensure diversity and performance standards.
